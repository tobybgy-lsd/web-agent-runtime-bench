# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Keep failure cases sanitized.
- Avoid posting secrets, private data, credentials, cookies, tokens, or private screenshots.
- Respect platform rules and authorization boundaries.

## Unacceptable Behavior

- Harassment or abusive comments.
- Sharing credentials or private user data.
- Requesting or providing bypass, evasion, credential abuse, or unauthorized access guidance.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct.
