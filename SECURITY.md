# Security Policy

Agent Failure Doctor is local-first. It does not upload artifacts, logs, traces, cookies, credentials, or network captures to any remote service.

## Before Sharing Failure Artifacts

Please remove:

- passwords
- cookies
- session tokens
- authorization headers
- API keys
- private URLs
- private data
- customer data
- personal information

## What This Project Does Not Support

This project does not provide CAPTCHA bypass, bot evasion, credential extraction, unauthorized scraping, anti-abuse circumvention, or guidance for bypassing access controls.

## Reporting Security Issues

Please do not open a public issue containing secrets. Open a minimal report without sensitive data, or contact the maintainer privately.
