# Android Ops Dashboard

The Local Web Console can expose a read-only Android Ops Center status with:

- Device Farm
- Device Health
- Appium Sessions
- Device Lease
- Recovery
- Business Templates
- Data Binding
- Scheduler
- Replay
- Flaky Detector
- Compatibility Report
- Mutation Guard
- Ops Metrics
- Runbook

Raw screenshots are hidden by default. Final submit and business mutation are blocked
by default.
