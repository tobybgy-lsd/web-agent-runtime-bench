# Composite Diagnosis Limits

The Composite Diagnosis P95 gate measures known and modeled automation failure
families. It is not a claim of universal reasoning.

Agent Failure Doctor may still return `insufficient_evidence` when a report only
contains a vague description, screenshot metadata, or conflicting weak signals.

The engine remains local-first and deterministic:

- no external AI calls
- no upload of user artifacts
- no raw secret retention
- no private challenge solution publishing
- no access-control circumvention guidance

For anti-bot or access-risk cases, the safe next action is to confirm
authorization, use an official API or authorized export, request manual review,
or stop automation when authorization is unclear.
