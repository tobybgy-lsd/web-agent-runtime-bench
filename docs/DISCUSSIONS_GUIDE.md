# Discussions Guide

GitHub Discussions cannot be enabled from the repository tree, but these categories are recommended when it is enabled.

## Failure Cases

Use this for sanitized logs, trace summaries, network snippets, screenshot metadata, and short failure descriptions.

## Ideas

Use this for feature requests, new failure categories, report improvements, and integration ideas.

## Show and Tell

Use this for examples of reports that helped debug Playwright, browser-use, RPA, crawler, or AI browser-agent failures.

Do not post credentials, cookies, tokens, authorization headers, private URLs, personal data, customer data, or private screenshots.
