# P98 Limits

P98 is controlled project maturity. It does not include ecosystem maturity.

P98 does not mean:

- universal business logic understanding
- bypass or evasion capability
- automatic repair of every source-code failure
- external adoption, stars, forks, PyPI downloads, or third-party production usage

Anti-bot risk remains detection, routing, and compliant next action only: reduce load, use official APIs, confirm authorization, perform manual verification, or stop if unauthorized.
