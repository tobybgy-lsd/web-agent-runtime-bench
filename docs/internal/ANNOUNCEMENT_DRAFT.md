# Announcement Draft

English title:

I built a local-first failure doctor for AI browser automation, Playwright, and crawler debugging

中文标题：

我做了一个本地优先的 AI 自动化失败诊断工具：支持 trace、log、network 和 Codex 修复 Prompt

Short post:

Agent Failure Doctor is a local-first tool that diagnoses failed AI browser automation, Playwright, crawler, and RPA runs from sanitized trace.zip, logs, network summaries, screenshot metadata, or user descriptions.

It generates diagnosis, evidence, next action, repair suggestions, a GitHub issue draft, and a Codex fix prompt.

I am not asking for stars. I am looking for sanitized failure samples.

欢迎提交脱敏后的失败日志、trace.zip、network.json 或错误描述。适合公开的案例可能会加入 validation corpus。

Please do not include credentials, cookies, tokens, authorization headers, private URLs, personal data, customer data, or private screenshots.
