# Manual GitHub About Update

Use this if repository metadata cannot be changed with the GitHub CLI.

1. Open the repository homepage:
   [https://github.com/tobybgy-lsd/web-agent-runtime-bench](https://github.com/tobybgy-lsd/web-agent-runtime-bench)
2. In the right-side About area, click the gear/edit button.
3. Set Description:

   ```text
   Local-first failure diagnosis, auto collection, repair planning, AI handoff, and verification for Playwright, crawler, RPA, and agent workflows.
   ```

4. Set Website:

   ```text
   https://pypi.org/project/agent-failure-doctor/
   ```

5. Add Topics:

   ```text
   agent
   ai-agent
   playwright
   crawler
   rpa
   debugging
   failure-diagnosis
   automation
   codex
   cursor
   claude-code
   pypi
   ```

6. Save changes.
