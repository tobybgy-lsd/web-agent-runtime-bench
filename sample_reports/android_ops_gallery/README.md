# Android Ops Gallery

This directory is a lightweight public gallery for v5.3.0 Android Ops reports.

The canonical generated validation report is:

- `validation/android_ops_validation.json`

Example input materials:

- `templates/android_device_matrix/mock_farm.json`
- `templates/android_business_flows/post_image_text_save_draft.yml`
- `examples/android_ops_cases/business_data_binding/`

The gallery intentionally contains no private device identifiers, screenshots, account data, credentials, or real business submission artifacts.
