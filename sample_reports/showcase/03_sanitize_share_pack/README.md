# Showcase 03: Sanitize Share Pack

v2.1 sample showing how a failed-run report can be converted into a conservative shareable pack.

## Input Summary

- Input type: local report folder
- Evidence: sanitized log summary and network summary
- Raw trace archive: not included

## Output

- redacted logs
- redacted network summary
- trace metadata only
- redaction report
- `safe_to_share=false`

## Suggested Next Action

Review the generated files manually before attaching the pack to an issue.
