# Plugin Templates

Use `failure-doctor plugin scaffold` to generate a complete local-only template.

Available template families:

- framework_adapter
- evidence_adapter
- diagnosis_rule
- industry_pack
- console_extension
- ci_extension
- kb_pattern
- reasoning_tool
- report_exporter
- validation_pack

Generated plugins include:

- `plugin_manifest.json`
- `plugin.py`
- `schemas/input.schema.json`
- `schemas/output.schema.json`
- `tests/test_plugin.py`
- `README.md`
- `FORBIDDEN_ACTIONS.md`
